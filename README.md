# Rinme's GitPage Source

### Made with [Vue.js](https://vuejs.org/) and [Vuetify.js](https://vuetifyjs.com/)

---

# Pre-require

- **Node.js** =< 11

- **npm**

- **yarn**

---

# Get started!

- **Install packages**

```zsh
yarn install
```

or

```zsh
npm install
```



- **Get it working**

```zsh
yarn serve
```

or

```zsh
yarn start
```



- Building

```zsh
yarn build
```
